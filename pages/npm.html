Node.js NPM

What is NPM?

NPM is a package manager for Node.js packages, or modules if you like.

www.npmjs.com hosts thousands of free packages to download and use.

The NPM program is installed on your computer when you install Node.js

What is a Package?

A package in Node.js contains all the files you need for a module.

Modules are JavaScript libraries you can include in your project.

Download a Package

Downloading a package is very easy.

Open the command line interface and tell NPM to download the package you want.

I want to download a package called "upper-case":

C:\Users\Your Name>npm install upper-case

Using a Package

After the package is installed, you can use it in your Node.js file:

var uc = require('upper-case'); http.createServer(function (req, res) { res.writeHead(200, {'Content-Type': 'text/html'}); res.write(uc.upperCase("Hello World!")); res.end(); }).listen(8080);

Package.json

The package.json file is the heart of Node.js system. It records important metadata about a project which is required before publishing to NPM, and also defines functional attributes of a project that npm uses to install dependencies, run scripts, and identify the entry point to our package.

Creating a Package.json

The package.json file is automatically generated when you run:

npm init

A wizard will help you create the file. Press Enter to accept the default values or provide your own:

package name: (my-project) version: (1.0.0) description: My first Node.js project entry point: (index.js) test command: git repository: keywords: author: license: (ISC)

Example Package.json

{ "name": "my-project", "version": "1.0.0", "description": "My first Node.js project", "main": "index.js", "scripts": { "test": "echo \"Error: no test specified\" && exit 1", "start": "node index.js" }, "keywords": [], "author": "", "license": "ISC", "dependencies": { "upper-case": "^2.0.2" } }

Common NPM Commands

Command Description
npm init Creates a package.json file
npm install Installs dependencies defined in package.json
npm install package-name Installs a specific package
npm install -g package-name Installs a package globally
npm install --save-dev package-name Installs a package as a development dependency
npm uninstall package-name Removes a package
npm update Updates all packages
npm list Lists installed packages

Dependencies vs DevDependencies

In your package.json file, there are two types of dependencies:

  • dependencies: Packages required for the application to run
  • devDependencies: Packages only needed for development
// Install as regular dependency npm install package-name --save // Install as development dependency npm install package-name --save-dev

Project Setup Example

Here's a complete example of setting up a new Node.js project:

# Create a new directory mkdir my-node-project cd my-node-project # Initialize npm project npm init -y # Install some dependencies npm install express npm install nodemon --save-dev # Add a start script to package.json { "scripts": { "start": "nodemon index.js" } } # Create index.js and start coding!
